# Outliner v0.1.0

Initial public release of Outliner: a keyboard-first self-hosted outline editor with Markdown import/export, sharing, public links, and SQLite persistence.

## Highlights

- Nested rows with keyboard editing, indentation, movement, collapse/expand, colours, and multi-select
- Markdown import/export for portable outline structure
- Full JSON backup import/export
- Email/password accounts with per-user list ownership
- Shared lists with viewer/editor roles
- Read-only public links
- Owner-only checkpoints and revision restore
- SQLite persistence with local startup cache
- Smoke, unit, and browser test coverage

## Requirements

- Node.js 24.3+
- npm

## Install

```bash
npm install
npm start
```

Open `http://127.0.0.1:4310` and create the first account.

## Known Limits

- Collaboration is not live real-time editing.
- Conflict handling is row-based.
- Public links are read-only.
- Public deployments should use HTTPS and regular SQLite backups.
