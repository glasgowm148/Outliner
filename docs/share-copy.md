## Longer Post

I built Outliner because I wanted a fast outline editor that I could self-host, back up as a SQLite file, and move in and out of Markdown without fighting a proprietary format.

It supports keyboard-first row editing, nested outlines, Markdown import/export, JSON backups, shared lists with viewer/editor roles, read-only public links, and revision checkpoints.

The stack is deliberately small: plain browser JavaScript, Node.js, and SQLite.

Repo: https://github.com/glasgowm148/Outliner
